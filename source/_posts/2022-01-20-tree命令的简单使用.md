## tree常用命令

|命令|描述|
|---|---|
|-a |显示所有文件|
|-d |仅显示目录|
|-L level|列出的目录的最大深度|
|-f|打印每个文件的完整路径前缀|
|-P pattern|列出匹配给定模式的文件|
|-I pattern|不列出匹配给定模式的文件|
|-o filename|输出到文件，而不是标准输出|
|-N|按原样打印不可打印的字符(比如 中文)|
|-J|打印输出树的JSON表示|

## 举个栗子

```bash
# 仅列出目录，最大目录深度为3，忽略node_modules和dist目录
tree -d -L 3 -I 'node_modules|dist'
```
## 最后

```
# 将列出的结果追加到 README.md 文件 ( 记得给内容加上```content```表示代码块 )
# 然后加上注释，就可以得到好看的项目目录结构
tree -I 'node_modules|dist|.git' -L 3 -a >> README.md
```
